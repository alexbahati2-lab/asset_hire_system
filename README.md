# Asset Hire Management System

A Django-based web application to manage the rental and hire of assets.  
Includes:

- Asset tracking
- Hire management
- Customer information
- Payment processing via M-Pesa Paybill (C2B)

---

## Features

- **Admin Dashboard**: Built-in Django admin with custom branding.  
- **Asset Management**: Add, view, and track asset status.  
- **Hire Management**: Assign assets to customers, track due dates, prevent double-assignment.  
- **Payments**: M-Pesa Paybill integration with automatic payment recording.  
- **Search & Filters**: Easily search for persons, assets, hires, and payments.  
- **Safe Deployment**: Ready for deployment on Render with `Procfile` and `requirements.txt`.
